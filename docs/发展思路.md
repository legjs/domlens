# MVP 方案（第一阶段）

## 产品定位

# Runtime Context Bridge for Claude Code

浏览器插件负责：

```text id="clwmu5"
运行态上下文
→ 自动同步给 Claude Code
```

Claude Code 负责：

```text id="clwmu6"
理解页面
→ 修改源码
→ 页面热更新
```

---

# 一、整体架构

```text id="clwmu7"
Browser Extension
    ↓
Runtime Inspector
    ↓
Source Mapper
    ↓
Local Bridge Server
    ↓
Claude Code MCP
    ↓
Claude 修改源码
    ↓
Vite/Next HMR
    ↓
Browser 自动更新
```
