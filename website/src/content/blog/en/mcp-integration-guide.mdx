## What is MCP?

The **Model Context Protocol (MCP)** is an open standard that enables AI tools to communicate with external services. DOM Context implements an MCP server that gives AI coding agents direct access to live DOM context.

## Prerequisites

- Node.js 18 or later
- Claude Code CLI installed
- DOM Context Chrome extension installed

## Step 1: Start the Runtime Server

```bash
git clone https://github.com/user/claude-dom-context.git
cd claude-dom-context
pnpm install
pnpm runtime:start
```

The server will automatically discover an available port in the range 4777-4787.

## Step 2: Configure Claude Code

Add the following to your Claude Code MCP configuration:

```json
{
  "mcpServers": {
    "dom-context": {
      "command": "node",
      "args": ["src/server/index.ts"],
      "type": "stdio"
    }
  }
}
```

## Step 3: Use in Your Workflow

1. Open any webpage in Chrome
2. Activate the DOM Context inspector (Alt + Hover)
3. Click or Ctrl+Click elements to capture context
4. In Claude Code, the AI can now call `get_latest_context` to retrieve the captured DOM data
5. Use `get_prompt` to get a ready-to-use Markdown description

## Available Tools

| Tool | Description |
|------|-------------|
| `get_latest_context` | Get the most recent DOM context capture |
| `get_prompt` | Get a formatted Markdown prompt for AI chat |
| `list_contexts` | List all stored context history |
| `clear_contexts` | Clear all stored context data |
| `get_user_prompt` | Get the latest prompt from the floating panel |

## Tips

- Use **multi-select** (Ctrl+Click) to capture multiple related elements
- The **context compression** engine automatically reduces verbose DOM data
- Use the **floating panel** to add custom notes alongside element data
- Context history stores up to 20 entries by default

## Troubleshooting

### Server won't start
Make sure ports 4777-4787 are not in use by other applications.

### Claude Code can't connect
Verify the path in your MCP configuration points to the correct `src/server/index.ts` file.

### Context is empty
Make sure you've captured at least one element by hovering or clicking with the inspector active.
