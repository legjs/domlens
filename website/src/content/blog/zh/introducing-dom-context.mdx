## DOM Context 是什么？

DOM Context 是一个 **Chrome 扩展**，为 AI 编程助手提供**运行时 DOM 上下文感知能力**。AI 工具不再依赖静态截图或手动描述，而是可以直接访问网页上任何元素的结构化实时信息。

## 为什么构建它

随着 AI 编程助手变得越来越强大，瓶颈已经从"AI 能写代码吗？"转移到了"AI 能理解页面上已有什么吗？"

传统方法要求开发者：

1. 截图并上传
2. 手动在提示中描述 DOM 结构
3. 复制粘贴 HTML 源代码
4. 口头解释 CSS 布局约束

DOM Context 自动化了所有这些操作。只需悬停或点击，扩展就能捕获全面的元素数据，并通过标准化的**模型上下文协议（MCP）**接口提供。

## 核心功能

### 元素检查器

悬停在任何 DOM 元素上，即可立即查看其标签、类名、尺寸、计算样式和布局属性。检查器提供可视化覆盖层，显示盒模型（外边距、边框、内边距、内容）。

### 多元素选择

按住 `Ctrl` 并点击可选择多个元素。每个选择自动标记为 A 到 J，并存储完整的上下文数据。非常适合向 AI 代理描述组件关系。

### MCP 集成

DOM Context 运行本地 MCP 服务器，暴露五个工具：

- `get_latest_context` — 获取最新的 DOM 捕获
- `get_prompt` — 获取格式化的 Markdown 提示
- `list_contexts` — 浏览捕获历史
- `clear_contexts` — 重置历史
- `get_user_prompt` — 获取浮动面板的提示

### 上下文压缩

内置压缩引擎将 DOM 上下文精简为最相关的信息，在不丢失重要结构细节的前提下，提高 AI 消费效率。

## 开始使用

1. 安装 DOM Context Chrome 扩展
2. 使用 `pnpm runtime:start` 启动本地运行时服务器
3. 在 AI 工具设置中配置 MCP
4. 开始检查元素，让 AI 工具完成剩下的工作
